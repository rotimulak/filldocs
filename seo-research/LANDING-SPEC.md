# FillDocs — спецификация одностраничника

> Задача в разработку. Сервис сверху (как есть), SEO-тексты ниже.

## Концепция

Одна страница, без роутера:

```
┌─────────────────────────────────────┐
│  Header (текущий, без изменений)    │
│  Две панели (текущие, без изменений)│
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │
│  Секция: Как это работает           │
│  Секция: Для кого                   │
│  Секция: Почему FillDocs            │
│  Секция: Сравнение                  │
│  Секция: FAQ                        │
│  Footer                             │
└─────────────────────────────────────┘
```

- Верхняя часть (header + панели + текущий footer-строка) — **не трогаем**
- Под ними — SEO-секции
- Все CTA-кнопки — `scroll to top` (к приложению)
- Без react-router-dom, без новых зависимостей

### 3.4. Comparison — «Как обычно vs. С FillDocs»

```
┌──────────────────────────────────────────────────────┐
│  bg-gray-50                                          │

│  │ ✗ Опечатки          │  │ ✓ Точно            │   │

│  └─────────────────────┘  └─────────────────────┘   │
```

**Тексты:**

**Колонка «Как обычно»:**
- Копируете ИНН, КПП, адреса из одного файла в другой вручную
- Ищете, в какую ячейку таблицы что вставить
- Тратите 30+ минут на один документ
- Допускаете опечатки и пропускаете поля
- Повторяете всё заново для каждого нового шаблона

**Колонка «С FillDocs»:**
- Загрузили файл — ИИ извлёк все реквизиты сам
- ИИ сам находит нужные ячейки в таблице
- Готовый документ за 30 секунд
- Точная подстановка данных без ошибок
- Один раз извлёк — заполняй сколько угодно шаблонов

**Дизайн:**
- Контейнер: `max-w-4xl mx-auto px-4 py-16`
- Две колонки: `grid grid-cols-1 md:grid-cols-2 gap-6`
- Левая: `bg-white rounded-xl p-6 border border-gray-200`
  - Заголовок: `text-gray-400 font-semibold text-sm uppercase tracking-wide`
  - Пункты: `text-gray-500`, иконка ✗ `text-red-400`
- Правая: `bg-blue-50 rounded-xl p-6 border border-blue-200`
  - Заголовок: `text-blue-600 font-semibold text-sm uppercase tracking-wide`

**Вопросы и ответы:**

| # | Вопрос | Ответ |
|---|--------|-------|
| 1 | Какие форматы файлов поддерживаются? | FillDocs работает с документами Word (.doc и .docx). Файлы .doc автоматически конвертируются в .docx перед обработкой. |
| 2 | Где хранятся мои данные? | Загруженные файлы обрабатываются и немедленно удаляются с сервера. Извлечённые реквизиты сохраняются в localStorage вашего браузера — они никогда не покидают ваше устройство и не передаются третьим лицам. |

| 5 | Что если ИИ заполнил поле неправильно? | FillDocs показывает отчёт: какие поля заполнены, какие пропущены. Скачайте документ, проверьте в Word и внесите правки. Рекомендуем всегда проверять результат перед отправкой. |
| 6 | Сколько шаблонов можно заполнить? | Неограниченно. Реквизиты сохраняются в браузере — загружайте столько шаблонов, сколько нужно. |
| 7 | Нужно ли что-то устанавливать? | Нет. FillDocs работает в браузере. Chrome, Firefox, Safari, Edge на любой ОС. |
