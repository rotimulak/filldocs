## 3. AI-инструменты для работы с документами

### 3.1 ChatGPT / Claude / YandexGPT

- Генерация текстов документов через AI
- Нет специализированного интерфейса для заполнения шаблонов
- Нет работы с форматированием .docx

### 3.2 AI Document Automation (Docsumo, Rossum, Instabase)

- AI-извлечение данных из документов (OCR + NLP)
- Международные продукты, не локализованы для РФ
- **Ключевое отличие от FillDocs:** они извлекают данные из документов (входящий поток), FillDocs заполняет шаблоны данными (исходящий поток)

### 3.3 ELMA365 AI

**URL:** https://elma365.com/ru/elma-ai/

- Корпоративная платформа с ИИ для документов
- Тяжёлое enterprise-решение

## 5. Анализ контент-стратегий конкурентов

### 5.1 Типы контента, которые публикуют конкуренты

**Контур (kontur.ru, Диадок):**
- Блог с экспертными статьями по бухгалтерии, налогам, законодательству
- Вебинары и видеоуроки
- Калькуляторы (налоги, взносы, пени)
- Справочные материалы по заполнению форм
- SEO-статьи формата «Как заполнить [название формы] в 2026 году»

**СБИС:**
- Новости ЭДО и отчётности
- Инструкции по работе с системой
- Кейсы клиентов
- Сравнения с конкурентами

**Документовед / FreshDoc:**
- SEO-оптимизированные шаблоны документов (каждый шаблон = отдельная страница)
- Статьи «Как составить [тип документа]»
- FAQ по типовым ситуациям

### 5.2 Темы, которые покрывают конкуренты

1. **Реквизиты и документооборот:**
   - «Как узнать реквизиты компании по ИНН»
   - «Какие реквизиты указывать в договоре»
   - «Обязательные реквизиты первичных документов»
   - «Как заполнить карточку контрагента»

2. **Шаблоны и заполнение:**
   - «Шаблон договора поставки 2026»
   - «Образец заполнения акта выполненных работ»
   - «Как заполнить товарную накладную»
   - «Анкета участника закупки: образец заполнения»

3. **Закупки (44-ФЗ, 223-ФЗ):**
   - «Документы для участия в тендере»
   - «Анкета участника закупки -- что заполнять»
   - «Реквизиты компании для госзакупок»

4. **ЭДО и автоматизация:**
   - «Как перейти на электронный документооборот»
   - «Преимущества ЭДО для бизнеса»

### 5.3 Структура посадочных страниц конкурентов

Типичная структура (Контур, СБИС, Документовед):

1. **Hero-секция:** заголовок с ценностным предложением + CTA «Попробовать бесплатно»
2. **Проблема:** описание боли (ручное заполнение, ошибки, потеря времени)
3. **Решение:** как продукт решает проблему (3-4 пункта с иконками)
4. **Как это работает:** пошаговый процесс (3-5 шагов с иллюстрациями)
5. **Преимущества:** сетка из 4-6 преимуществ
6. **Тарифы:** таблица тарифов или CTA для получения демо
7. **Отзывы/кейсы:** логотипы клиентов, цитаты
8. **FAQ:** частые вопросы
9. **CTA-финал:** повторный призыв к действию

## 6. Ключевые выводы

### 6.1 Уникальное позиционирование FillDocs

| Возможность | Конкуренты | FillDocs |
|-------------|-----------|----------|
| Извлечение реквизитов из произвольного .docx | Нет (только по ИНН из ЕГРЮЛ) | Да (AI из любого документа) |
| Заполнение произвольного шаблона .docx | Нет (только свои формы) | Да (любой Word-файл) |
| Не нужна интеграция с 1С/ERP | -- | Да |
| Не нужна регистрация/подписка (MVP) | Нет | Да |
| Работа с нестандартными формами (закупки) | Нет | Да |

### 6.2 Позиционирование относительно конкурентов

```
Контур, СБИС, 1С:
  "У нас экосистема для всего бизнеса"
  → Тяжёлые, дорогие, нужна интеграция

Документовед, FreshDoc:
  "У нас готовые шаблоны документов"
  → Только свои шаблоны, нельзя загрузить свой

DaData:
  "API для разработчиков"
  → Нужен программист для интеграции

FillDocs:
  "Загрузите два Word-файла -- получите заполненный документ"
  → Простота, ИИ, любой шаблон, без интеграции
```

### 6.3 Главное конкурентное преимущество

Конкуренты работают по принципу: **ИНН → ЕГРЮЛ → фиксированный набор полей → свои шаблоны**.

FillDocs работает по принципу: **любой документ с реквизитами → AI → любой шаблон**.

Это принципиально другой подход, идеально решающий задачу тендерных специалистов.

### 6.4 Рекомендуемые ценностные предложения

1. **«Любой шаблон Word -- не только наши формы»** -- главное отличие от Документоведа, FreshDoc, СБИС
2. **«ИИ извлекает реквизиты из любого документа»** -- не нужно вводить вручную или знать ИНН
3. **«Без 1С, без регистрации, без подписки»** -- барьер входа минимальный
4. **«30 секунд вместо 30 минут»** -- конкретная экономия времени
5. **«Работает с нестандартными формами»** -- закупочные анкеты, формы контрагентов
6. **«Данные не сохраняются на сервере»** -- безопасность, актуально для бизнеса
