## 4. Анализ поискового намерения (Search Intent)

### 4.1. Информационное намерение (Informational)

Пользователи ищут информацию о том, как решить свою проблему:

- **"как автоматически заполнить документ Word реквизитами"** -- ищут инструкцию / способ
- **"как заполнить анкету участника закупки"** -- ищут пошаговое руководство
- **"как подставить реквизиты в шаблон документа"** -- ищут технический метод
- **"как извлечь реквизиты компании из документа"** -- ищут инструмент или способ

**Рекомендация для контента:** Создать обучающие статьи в блоге, которые отвечают на эти вопросы и подводят к использованию FillDocs как решения.

### 4.2. Навигационное намерение (Navigational)

Пользователи ищут конкретные сервисы:

- **"Doczilla"**, **"Комбинатор"**, **"FreshDoc"**, **"DocWebService"** -- конкуренты
- **"1С:Контрагент"** -- сопутствующий сервис
- **"конструктор документов Гарант"** -- конкурент

**Рекомендация:** Создать страницы сравнения "FillDocs vs [конкурент]" для перехвата этого трафика.

### 4.3. Транзакционное намерение (Transactional)

Пользователи готовы использовать сервис:

- **"заполнить документ онлайн бесплатно"**
- **"генератор документов по шаблону бесплатно"**
- **"конструктор документов онлайн"**
- **"заполнить карточку предприятия по ИНН"**
- **"скачать заполненный документ Word"**

**Рекомендация:** Оптимизировать лендинг и главную страницу под эти запросы; подчеркивать бесплатность / простоту.

### 4.4. Коммерческое намерение (Commercial Investigation)

Пользователи сравнивают решения перед выбором:

- **"лучший конструктор документов"**
- **"сравнение конструкторов документов"**
- **"doczilla или комбинатор что лучше"**
- **"программа для заполнения документов обзор"**

**Рекомендация:** Создать контент типа "Топ-N сервисов для заполнения документов" с включением FillDocs.

## 6. Конкурентный ландшафт (обзор)

### Прямые конкуренты (конструкторы документов)

| Конкурент | Сильные стороны | Ключевые слова, по которым ранжируется |
|---|---|---|
| **Doczilla.pro** | AI-ассистент, полный цикл договоров, интеграции | конструктор документов, автоматизация договоров |
| **Комбинатор** | Простота, интеграция с CRM/1С, морфология | конструктор документов и договоров, шаблонизатор |
| **FreshDoc** | Библиотека шаблонов, простота для новичков | конструктор договоров, шаблоны договоров |
| **DataDoc** | Комплекты документов, ссылки для контрагентов | конструктор договоров, генератор документов |
| **DocWebService** | Бесплатность, множество форматов | заполнение документов онлайн бесплатно |

### Косвенные конкуренты

| Конкурент | Пересечение |
|---|---|
| **1С:Контрагент** | Заполнение реквизитов по ИНН |
| **Битрикс24** | Умные реквизиты, генератор документов |
| **kardx.ru / kartochka.info** | Генерация карточки предприятия |
| **Excel + Word (слияние)** | Ручной метод, часто ищут |

### Уникальное позиционирование FillDocs

Ни один из конкурентов не предлагает точно такой же подход:
1. **Загрузка произвольного документа** с реквизитами (не ввод вручную и не поиск по ИНН)
2. **ИИ-извлечение** реквизитов из загруженного файла
3. **ИИ-заполнение** произвольного шаблона Word (не только фиксированные шаблоны с метками)

Этот подход идеально решает задачу тендерных специалистов, которым нужно заполнить чужой шаблон (анкету заказчика) своими реквизитами.
