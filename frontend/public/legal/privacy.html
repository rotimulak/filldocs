3. Термины и определения

3.1. Персональные данные — любая информация, относящаяся к прямо или косвенно определённому или определяемому физическому лицу.

3.2. Обработка персональных данных — любое действие с персональными данными, включая сбор, запись, хранение, использование, передачу, удаление.

3.3. Оператор — лицо, организующее и осуществляющее обработку персональных данных.

3.4. Сервис — веб-приложение FillDocs, предоставляющее услуги по автоматическому заполнению реквизитов в документах Word с помощью искусственного интеллекта, доступное по адресу https://filldocs.ru.

4. Перечень собираемых данных

4.1. Сервис FillDocs спроектирован по принципу минимизации сбора данных.

4.1.1. Загруженные документы

  • Файлы, загруженные Пользователем, обрабатываются на сервере и немедленно удаляются после завершения операции.
  • Заполненные документы удаляются сразу после скачивания.
  • Оператор НЕ хранит копии загруженных или заполненных документов.

4.1.2. Извлечённые реквизиты

  • Реквизиты, извлечённые из документов, хранятся исключительно в localStorage браузера Пользователя.
  • Реквизиты НЕ передаются на сервер Оператора для хранения и никогда не покидают устройство Пользователя (за исключением момента заполнения шаблона).

4.1.3. Данные при оплате

  • При добровольной оплате (донате) Пользователь перенаправляется на страницу платёжной системы ЮKassa.
  • Оператор НЕ собирает и НЕ хранит данные банковских карт, номера счетов и иную платёжную информацию.
  • Все платёжные данные обрабатываются исключительно платёжной системой ЮKassa в соответствии с её собственной политикой конфиденциальности.

4.2. Оператор НЕ собирает и НЕ хранит:

  • IP-адреса пользователей
  • Файлы cookie
  • Данные для регистрации (регистрация не требуется)
  • Контактные данные пользователей
  • Историю использования сервиса

5. Цели обработки данных

5.1. Оператор обрабатывает данные исключительно для:

  • Оказание услуг — извлечение реквизитов из документов и заполнение шаблонов

5.2. Оператор НЕ использует данные для маркетинга, профилирования, передачи третьим лицам в рекламных целях.

6. Правовые основания обработки

6.1. Правовым основанием обработки данных является:

  • Исполнение договора — обработка документов осуществляется в рамках оказания услуг по договору оферты
  • Согласие Пользователя — загрузка файла в Сервис является выражением согласия на его обработку

7. Хранение данных

7.1. Загруженные файлы: обрабатываются в оперативной памяти и во временных файлах, удаляются немедленно после обработки.

7.2. Заполненные документы: хранятся на сервере только до момента скачивания, затем автоматически удаляются.

7.3. Реквизиты: хранятся в localStorage браузера Пользователя — Оператор не имеет к ним доступа.

7.4. Серверы: расположены на территории Российской Федерации (хостинг-провайдер: AdminVPS).

8. Передача данных третьим лицам

8.1. Оператор НЕ передаёт персональные данные Пользователей третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

8.2.1. Платёжная система (ЮKassa)

Оператор НЕ передаёт персональные данные в платёжную систему. Взаимодействие осуществляется через создание счёта на оплату. Пользователь самостоятельно вводит платёжные данные на стороне ЮKassa.

8.2.2. Провайдер нейросетей (Anthropic)

Для извлечения реквизитов и заполнения документов используются сервисы искусственного интеллекта. Текст документов передаётся в API провайдера нейросетей для обработки.

8.2.3. Хостинг-провайдер (AdminVPS)

Сервис размещён на серверах провайдера в рамках договора на предоставление хостинг-услуг.

9. Трансграничная передача данных

9.1. В процессе оказания услуг может осуществляться трансграничная передача текста документов при использовании API нейросетей (Anthropic).

9.2. Передаётся только текстовое содержимое документов для обработки. Оператор не контролирует, какие персональные данные могут содержаться в документах, загруженных Пользователем.

10. Защита данных

10.1. Оператор принимает необходимые меры для защиты данных:

  • Шифрование при передаче: все данные передаются по защищённому протоколу HTTPS
  • Немедленное удаление: файлы не хранятся на сервере после обработки
  • Ограничение доступа: доступ к серверам имеют только владельцы Сервиса

11.1. Пользователь имеет право:

  • Получить информацию о своих данных, обрабатываемых Оператором
  • Запросить удаление данных из localStorage (выполняется самостоятельно через очистку данных браузера)

11.2. Для реализации своих прав Пользователь может обратиться к Оператору через Telegram: @rotimulak

11.3. Оператор рассматривает обращения в течение 30 дней с момента получения.
